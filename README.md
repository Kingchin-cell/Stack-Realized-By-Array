📚 数组实现栈
 
参照《数据结构与算法分析》教材核心思想，基于静态数组手写实现栈结构，遵循先进后出（LIFO）原则，纯底层代码实现，无STL依赖，夯实数据结构基础。
 
✨ 项目简介
 
本项目采用**顺序存储结构（数组）**实现栈，通过数组模拟栈的存储空间，配合栈顶指针完成核心操作，完美还原教材中顺序栈的实现逻辑，理解顺序存储与栈结构的结合原理，适合数据结构入门学习。
 
🧱 核心特性
 
- 基于静态数组实现，结构简单、存取效率高
- 严格遵循栈**先进后出（LIFO）**操作规则
- 设定固定栈容量，可灵活修改容量大小
- 纯手写底层逻辑，代码简洁、注释完整
- 无第三方依赖，贴合教材知识点，便于课后复习
 
🎯 实现功能
 
栈初始化（设定数组容量、初始化栈顶指针）
入栈操作（push）
出栈操作（pop）
获取栈顶元素
判断栈是否为空/栈是否满
遍历栈内所有元素
清空栈、重置栈顶指针
获取栈内有效元素个数
 
🚀 编译运行
 
1. 开发环境：Visual Studio 2022 / 任意C/C++编译器
2. 直接编译源码文件，运行主程序即可
3. 控制台交互式测试栈的各项核心操作
